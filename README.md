Transforming a Pioneer CDJ100S into a USB Controller for Any DJ Application

Using a Blackpillar STM32, an ST-Link V2 recorder is required (I've included photos of it in the Pictures folder).

Remove the CD drive, bottom panel, and power supply.

Keep only the board containing the buttons, LCD, jog wheel, and pitch control.

The LCD was removed and replaced with a 16x2 LCD.

All buttons are functional, and only one USB cable is used for computer connection. To use it with Serato Pro, a mixer that enables Serato Pro is required. In my tests I used the DJM S11 mixer and CDJs for deck control.
